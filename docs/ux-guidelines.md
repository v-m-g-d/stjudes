# UX Guidelines (MVP)

## Principles
- Keep language plain and welcoming.
- Prioritize readability and simple navigation.
- Ensure all key actions are clear on mobile.

## Page-level intent
- Home: orient new visitors quickly.
- Forum: make thread reading and commenting obvious.
- News: highlight recent updates first.
- Plans: show each plan status and latest update date.
- Admin: minimize clicks for publish/moderate actions.

## Accessibility baseline
- Use semantic headings in order.
- Ensure keyboard navigation for links and buttons.
- Provide alt text for meaningful images.
- Maintain sufficient color contrast for text and controls.
